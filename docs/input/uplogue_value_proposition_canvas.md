# Uplogue -- Value Proposition Canvas

## 顧客側

-   **顧客のジョブ:**
    -   旅行の思い出を後から振り返れる形で残したい
    -   家族や友人に旅の流れを分かりやすく共有したい
    -   面倒な整理作業なしで記録を保管したい
-   **顧客のペイン:**
    -   写真はあるが時系列や場所が分からなくなる
    -   アルバム整理や文章化が手間で放置してしまう
    -   SNS投稿では旅の流れが伝わらない
-   **顧客のゲイン:**
    -   何もしなくても旅行の流れが分かる記録が欲しい
    -   見返した時に記憶が鮮明に蘇る
    -   説明不要で共有できる分かりやすい形

------------------------------------------------------------------------

## 提供価値側

-   **製品/サービス:**
    写真の位置情報と撮影時刻から旅行の行程を自動生成するAI旅行記録アプリ

-   **ペインリリーバー:**

    -   GPSと時刻から自動で並べ替え・整理
    -   スポット単位に自動グルーピング
    -   手入力・文章作成を不要にする

-   **ゲインクリエイター:**

    -   地図とタイムラインで旅を可視化
    -   代表写真により思い出がすぐ蘇る
    -   共有用フォーマットを自動生成

------------------------------------------------------------------------

## フィットコメント

手間なく記録したいというジョブに対し「自動整理」という中核価値が直接対応しており、作業負担のペインを除去しながら"振り返れる喜び"というゲインを同時に生む構造になっている。
