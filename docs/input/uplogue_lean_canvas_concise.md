# Uplogue -- Lean Canvas

**顧客セグメント**\
写真は撮るが記録や共有が続かない旅行者、体験重視の旅行者、カップル・家族の思い出保存層、文章作成が苦手なライトクリエイター。初期は「旅行後に写真が溜まって放置される人」をアーリーアダプターとする。

**顧客の課題**\
①旅行の記録を残したいが面倒で続かない\
②SNS投稿は作業になり、感情や文脈が残らない\
③旅行計画がレビュー依存で「自分らしい旅」にならない

**独自の価値提案**\
「あなたの思い出が、誰かの次の旅になる。」\
写真をアップするだけで意味のあるストーリーと知識に変換し、記録と発見を同時に成立させる"体験記憶プラットフォーム"。

**ソリューション**\
AIが写真・位置・時間から旅行ストーリーを自動生成し、タイムラインとして蓄積。\
その集合知から、他ユーザーの旅の発見や次の旅の着想を得られる。

**チャネル**\
SNSシェア導線、旅行後リマインド通知、旅行コミュニティ、宿泊施設・航空・観光連携、口コミ。

**収益の流れ**\
プレミアム生成サブスク、AI生成クレジット課金、保存容量拡張、観光事業者向けインサイト提供。

**コスト構造**\
AI推論コスト、ストレージ、開発人件費、ユーザー獲得費。

**主要指標**\
旅行後生成率、月間ストーリー生成数、シェア率、再利用率、データ増加速度。

**圧倒的な優位性**\
人間の体験文脈付き旅行データの蓄積により、ネットワーク効果とスイッチングコストを同時に生む。
