# Uplogue -- Customer Journey Map

**ペルソナ:**\
29歳男性・都市部勤務の会社員。旅行が好きで毎回大量に写真を撮るが、帰宅後は忙しく記録を放置しがち。

**サービス:**\
写真をアップロードするだけで旅行の思い出が自動的にストーリー化され、他の人の旅の発見にもつながるAI旅行記録アプリ。
