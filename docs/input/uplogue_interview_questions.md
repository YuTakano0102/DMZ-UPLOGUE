# Uplogue -- UX Interview Question Guide

**事業コンセプト:**\
写真をアップロードするだけで旅行の記録が自動的にストーリー化され、他の人の旅の発見にもつながるAI旅行記録サービス。

**想定顧客:**\
都市部に住む20〜30代の社会人。旅行が好きで写真は多く撮るが、記録や投稿は続かないライトユーザー。

**インタビューの目的:**\
旅行後の記録・共有・振り返りに関する実際の体験や感情を理解し、課題と期待価値を特定する。

------------------------------------------------------------------------

## 質問リスト

1.  最近行った旅行について、帰宅後にどんなことをしましたか？その流れを時系列で教えてください。

2.  旅行中に写真を撮るとき、どんな気持ちで撮っていますか？後で何をしたいと思って撮っていますか？

3.  旅行後に「記録を残したい」と思う瞬間はありますか？どんな場面ですか？

4.  写真や思い出を整理しようとしてやめてしまった経験があれば、具体的に教えてください。

5.  現在、旅行の思い出をどのように保存・共有していますか？満足している点と不満な点は何ですか？

6.  SNS投稿やアルバム作成をする際、面倒だと感じる瞬間はどこですか？

7.  過去の旅行を振り返るとき、どんな情報が残っていると嬉しいですか？逆に無くて困るものはありますか？

8.  他の人の旅行体験を見るとき、どんな内容だと参考になりますか？レビューとの違いは感じますか？

9.  「自動で旅行記録が作られる」と聞いたとき、便利そうだと思う点と不安に感じる点は何ですか？

10. 理想的な"思い出の残し方"があるとしたら、どんな体験になっていてほしいですか？
