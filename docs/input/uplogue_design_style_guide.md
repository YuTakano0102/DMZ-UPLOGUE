# Uplogue -- デザインコンセプト & 基本スタイルガイド

**サービス名:** Uplogue\
**サービス概要:**
複数写真の位置情報と時刻情報から旅の行程を自動生成する旅行記録アプリ\
**ブランドイメージ要素:**
記憶の再体験、静かな感動、信頼できる整理、生活に溶け込む知性\
**競合や参考デザイン:** Google
Photos（整理された信頼感）、Polarsteps（旅の記録性）、MUJIアプリ（静けさと余白）

------------------------------------------------------------------------

## 1. デザインコンセプトのキーワード

「記憶を整える」「静かな旅情」「時間のレイヤー」「説明不要の理解」「生活に溶ける知性」の5つを核とする。\
このサービスは"共有のための派手さ"ではなく、"思い出を再体験するための落ち着き"が価値の中心であるため、感情は強く動かしつつも過剰に主張しないトーンが必要になる。ユーザーは能動操作より閲覧体験を重視するため、UIは「気づいたら理解できる」透明性を目指す。

------------------------------------------------------------------------

## 2. ビジュアルスタイルの方向性

Look & Feelは「デジタル日記帳と地図の融合」。\
画面は白背景をベースに、写真と地図が主役となる構成としUIは極力存在感を抑える。余白を広く取り、情報密度を下げることで"旅を眺める"感覚を作る。アニメーションは速さではなく"間"を重視し、スクロール時に時間の流れを感じる微細なフェードを採用する。

------------------------------------------------------------------------

## 3. カラーパレット

-   **Soft Ivory #F6F5F2**:
    紙の日記のような温かい背景色。記録体験に安心感を与える\
-   **Deep Ocean #1E3A5F**:
    メインカラー。信頼感と知性を表現し地図との相性も良い\
-   **Travel Blue #3A7BD5**:
    操作可能要素。行動導線を明確にするが主張しすぎない\
-   **Sunset Amber #F2A65A**:
    ハイライトや思い出の強調。感情的な温度を付加する\
-   **Mist Gray #9AA4AE**:
    補助情報。UIを静かに保ちながら可読性を確保する

全体として「静かなベース +
感情のアクセント」で構成し、SNS的な高彩度は避ける。

------------------------------------------------------------------------

## 4. タイポグラフィ

日本語は可読性の高いヒューマニスト系サンセリフを使用し、本文は中太ウェイトで長文閲覧を前提とする。見出しはやや細めのウェイトで余白を広く取り、落ち着きと知的さを演出する。英数字はやや幅広の書体を用いて時間・日付・距離の情報を読み取りやすくする。

------------------------------------------------------------------------

## 5. UI要素のスタイル例

ボタンは角丸の小さいラウンド（8px前後）とし、カードUIと統一感を持たせる。シャドウは極めて弱く、浮遊感ではなく"紙の重なり"の表現に近づける。アイコンは線画ベースのミニマルスタイルで統一し、写真より目立たない濃度に調整する。インタラクションはホバーよりもフェードやスライドで時間的連続性を感じさせる。
